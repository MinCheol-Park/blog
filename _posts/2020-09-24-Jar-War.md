## J2EE 아카이브 유형 

### JAR (Java ARchive)
- 여러 자바 클래스파일 + 리소스 모음 
- 자바플랫폼에 어플리케이션 또는 라이브러리를 `배포하기 위한 파일 포맷`

<br/>
<br/>

## WAR (Web application ARchive)
- 클래스, JSP, 서블릿, XML, HTML 등등 Web Application을 이루는 자원을 한데 모아 배포 하는 `JAR 파일`.
- `Web Server` 혹은 `Web Application Server` 에서 실행 가능
