## 1. 파일 작성

```jshelllanguage
$ vi hello.sh
```
```jshelllanguage
#!/bin/bash
echo "Hello, World"
```
<br/>

## 2. 실행 권한 추가 

```jshelllanguage
$ chmod +x hello.sh
```
<br/>

## 3. 실행 

```jshelllanguage
$ ./hello.sh

Hello, World
```
